For this project, we have implemented a client application using a Python framwork which is called Dash to provide the ability for the users access the data of trending videos in differnt countires according to the following properties: 

	Analyze which factors will affect the popularity of the YouTube video by static and dynamic data min-ing. 
	Provide import data tools to insert the latest data from Kaggle dataset into MySQL.
	Display the popular trends of YouTube videos in dif-ferent regions. 
	Provide a play window to browse the latest popular video directly. 
	Predict the success probability according to the users' video attributes.
